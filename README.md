# Compteur de mots en Java

## Description

Ce programme Java permet de compter le nombre total de mots contenus dans un fichier texte.

## Fonctionnement

* Le programme lit un fichier ligne par ligne
* Chaque ligne est découpée en mots
* Les mots sont comptés puis additionnés
* Le total est affiché à la fin

## Prérequis

* Java 21 ou version compatible
* Un fichier texte nommé `TestExercice3.txt`
* 
## Utilisation

1. Placer le fichier `TestExercice3.txt` dans le même dossier que le programme
2. Compiler le programme 

## Erreurs possibles

* Si le fichier n’est pas trouvé, vérifier :

  * Le nom du fichier
  * Son emplacement
  * Le chemin utilisé dans le code
